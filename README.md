## Introduction
codebox是一个基于文件文件储存（每一份代码在`pool/`目录内有一份代码文件 QAQ ）的pastebin，使用`highlight.js`实现代码高亮、`clipboard.js`实现访问剪贴板。
## Features
+ 修改pastebin中的代码
+ 一键复制（支持`ios Safari`）
+ 夜间（深色）模式
## Usage

```shell
git clone https://github.com/pluto0x0/codebox.git
```

## Tips

如果无法写入文件，请设置 `save.php` 和其所在目录的权限
